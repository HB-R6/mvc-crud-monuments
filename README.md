# HB R6 - MVC PDO CRUD

## Configuration

### Base de données

Créer un fichier `config/db.ini` avec les paramètres de configuration :

```ini
DB_DRIVER=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_NAME=db_name
DB_CHARSET=utf8mb4
DB_USER=user
DB_PASSWORD=pass
```

> [!NOTE]  
> Vous pouvez utiliser le fichier `config/db.ini-template` comme modèle :
> `cp config/db.ini-template config/db.ini`
